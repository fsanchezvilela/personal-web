# My Personal Website PWA using Next + Typescript

## Getting Started

First Install Dependencies:
```bash
pnpm init
```

Commands For Development:
```bash
pnpm run dev
```

Command for Build:
```bash
pnpm run build
```

## Deploy on Vercel
